Getting Started
Deploy the Hubzz UI Kit in minutes.
Install
npm install
npm run build
npm run minify:css
npm run minify:js
Include

Reference the compiled assets:

<link rel="stylesheet" href="/css/hubzz.min.css">
<script src="/js/hubzz.min.js"></script>

Initialization is automatic for the helper scripts.

Customize
  • Edit Sass in src/hubzz-web-styles/
  • Rebuild with npm run build
  • Deploy to Vercel: vercel
