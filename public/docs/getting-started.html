Getting Started

Deploy the Hubzz UI Kit in minutes.

Install

npm install --save-dev esbuild cssnano

Build locally:

npm run build && npm run minify:css && npm run minify:js

Include via CDN

<link rel="stylesheet" href="/css/hubzz.min.css">
<script src="/js/hubzz.min.js"></script>

Initialize

Initialization is automatic. Use helpers as needed:

Hubzz.theme.toggle();
Hubzz.modal.open('#example');
Hubzz.tabs.activate('tab-1');
